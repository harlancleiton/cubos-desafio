# Cubos Backend

## API REST

##### Instruções (yarn)

1. Copiar .env.example para .env e definir variáveis

```bash
DB_DATABASE=db.json
```

2. Instalar dependências

```bash
yarn install
```

3. Iniciar servidor

```bash
yarn dev
```

4. Executar testes

```bash
yarn test
```

5. Build e Start

```bash
yarn build
yarn start
```

###### OBS: As requisições devem ser feitas na porta 3333

URL base: http://localhost:3333/api/v1
